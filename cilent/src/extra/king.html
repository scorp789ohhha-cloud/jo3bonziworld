Kings

Kings are moderators of BonziWORLD, being able to temp ban, kick, nuke, bless, delete messages, and cancel other user's TTS. You can spot them wearing a king (or other mod exlusive) hat.

Kings can login for a session using /godmode [password], login on their computer via a cookie with /pgodmode [password], logout on all computers with /logout, or completely remove their kingword with /godlock. If a kingmode password ever to leak, login and run this command (preferably in a private room), and everyone using it will be logged out. Godlock only had to be used once in BonziWORLD's history.
